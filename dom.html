Why dom

We use DOM to make connection between html-css and javascript

We can use js on top or in between Object or after everything

What is DOM?

Dom ==> Document Object Model
It means Dom Converts the whole HTML and CSS into Javascripts and execute its code.

Lets use

document.getElementsByTagName('tag') ==> returns a html list
document.getElementById('#id') ==> returns a specific node
document.getElementByClassName('.class') ==> returns a NodeList
document.querySelector('class tag') ==> returns the first node
document.querySelectorAll('class tag') ==> returns NodeList
